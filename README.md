## ✨ Features

- **Responsive layout** – works on desktop, tablet, and mobile
- **Clean typography** – uses Google Fonts (Patrick Hand + Inter)
- **No dependencies** – pure HTML & CSS, zero build step
- **Sections included:**
  - Navigation bar with brand logo
  - Hero section with title
  - Projects list
  - Work Experience
  - Open Source contributions
  - Education
  - Testimonials / Reviews
  - Footer

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AbdulrahmanArafat/personal-website-Template.git
   ```
2. Open `index.html` in your browser — no server required.

## 🛠️ Customization

- Replace `me.jpg` with your own photo.
- Edit the name, title, and content directly inside `index.html`.
- Adjust colors and fonts in the `<style>` block inside the `<head>`.
